1 Авторизация

API ключи
При первом запуске нажмите «Настроить API ключи». Получите api_id и api_hash на my.telegram.org → API development tools. Введите их в окне настроек.
Вход
Введите номер телефона в международном формате (+79991234567), получите код в Telegram, при необходимости введите 2FA-пароль.
Хранение секретов
Все ключи и сессия сохраняются в системном Keyring (Связка ключей на macOS, Credential Manager на Windows). В файлах конфигурации секреты не хранятся.
